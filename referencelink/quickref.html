    Getting Started

    Develop Setup

    • Clone Repository View on Github shell script $ git clone https://github.com/Fechin/reference.git
    • Install Dependencies in the project directory shell script $ npm install
    • Start a Dev Server http://localhost:4000 shell script $ npm run dev
    • Create or modify source/_posts/.md
    • Send us pull request and chill

    It's a good practice to refer to the source code of the CheatSheets reference.
